# zorro-docs-maker

1. read `z-*.html` code comment
    * put comment into entries
    * convert markdown to html
    * find `@element`, `@attribute`, `@property`, `@method`, `@event`
2. read `demo.html` code samples
    * find `section[primary]`, `[title]`, `[demo]`
3. generate docs
    * tag name
    * description
    * demo + code snippets
    * more description
        * attribute
        * property
        * method
        * event
